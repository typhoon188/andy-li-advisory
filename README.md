## Deployment

This is a static website. It can be deployed on:

- Netlify
- GitHub Pages
- Cloudflare Pages
- Any static hosting service

For Netlify, upload the full project folder so `index.html`, `styles.css`, and the `assets/` folder stay together.

For GitHub Pages:

1. Push this repository to GitHub.
2. Open repository settings.
3. Go to Pages.
4. Set source to the main branch and root folder.
5. Save and wait for the GitHub Pages URL.

## Maintenance Notes

Update `index.html` for content changes and `styles.css` for visual styling.

If changing the public website, redeploy the full folder to Netlify or push the updated repository to GitHub Pages.

Recommended next updates:

- Replace the temporary Netlify subdomain with a custom site name or personal domain.
- Add a downloadable executive CV when the final CV version is ready.
